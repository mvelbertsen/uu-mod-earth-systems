solver.dataStructures.ViscBox¶

class solver.dataStructures.ViscBox(*args, **kwargs)¶

Bases: ViscBox

Object containing parameters for using the optional high viscosity box.

If not required, instantiate in the Parameters object with ViscBox(0).

If required, instantiate with ViscBox(1) and then manually set the other parameters within the Parameters object.

Attributes¶

useINT

Flag which indicates if the model is using the high viscosity boxb feature.

xsizeFLOAT

x direction physical size of the high viscosity box, used with internal velocity wall to drive subduction.

ysizeFLOAT

y direction physical size of the high viscosity box, used with internal velocity wall to drive subduction.

xposFLOAT

relative distance along the box where the wall is positioned.
