solver.physics.markers_fns¶

Functions that relate to marker physics, including interpolation of quantities to/from the grid, marker viscosity calculation, subgrid stress/diffusion

Lower level functions for marker operations, including getting nearest node distances, interpolating a value to/from markers, etc. are in markerUtils.

Functions

advectMarkers(markers, grid, xnum, ynum, ...)

Displaces the markers using the calculated grid velocities for the current timestep.

gridToMarker(fields, markers_fs, markersx, ...)

Interpolates grid values to markers for a specified list of variables

markerViscosity(markers, materials, m, grid, ...)

Calculates the viscosity of a marker based on its material type.

markersToGrid(markers, materials, grid, ...)

Reads or calculates the marker values for density, thermal quantities, viscosity, stresses, mu and interpolates them to the grid.

subgridDiffusion(grid, markers, params, ...)

Calculates the subgrid diffusion temperature changes for all grid nodes.

subgridStressChanges(markers, grid, xnum, ...)

Calculates the subgrid stress changes, stored in grid.dsigxx, grid.dsigxy

updateMarkerErat(markers, materials, grid, ...)

Updates the marker eii/eii_grid ratio values
