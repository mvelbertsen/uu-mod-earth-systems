output.visualisation.plotTemperature¶

output.visualisation.plotTemperature(grid, params, ntstp, t_curr, xlims, ylims, aspect_ratio=None, height=None, plot_vel_arrows=False, arrow_stp=5)¶

Plot a single variable as a colormap, here it is the temperature but this serves as a template for making simple plots of simulation output. It includes the option to annotate the field with arrows showing velocity.

Parameters¶

gridGrid object

Grid containing all simulation variables.

paramsParameters object

Simulation’s parameters object.

ntstpINT

Current timestep number.

t_currFLOAT

Current time (s).

aspect_ratioFLOAT (OPTIONAL)

Option to manually set the aspect ratio of the plots.

heightFLOAT (OPTIONAL)

Option to manually set height of an axis.

vel_arrowsBOOL (OPTIONAL)

Flag indicating if velocity arrows should be added to the plot. Defualt is False.

arrow_stpINT (OPTIONAL)

Number of grid points between each velocity arrow. Default is 5.

Returns¶

None.
