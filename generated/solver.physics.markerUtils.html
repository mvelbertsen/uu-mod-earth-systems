solver.physics.markerUtils¶

Low level functions for marker calculations, including marker-to-grid/grid-to-marker interpolation, and finding nearest node distances.

Functions

applyGridContrib(field, xn, yn, dxm, dym)

Applies the contributions for the surrounding grid nodes to a marker for a given variable.

applyGridWeights(xnum, ynum, grid, grid0)

Apply the weighting to the averaging of the grid values, to complete the interpolation from markers in markersToGrid.

applyMarkerContrib(field, m_field, dxm, dym, ...)

Applies the contribution to given field from a marker to the surrounding nodes based on the distance to the nodes.

findNearestNode(gridx, gridy, xnum, ynum, ...)

Find the index of the nearest top-left basic node.

getMarkerNodeDistances(markerx, markery, ...)

Finds the distance to the nearest top-left node of a specified type for a given marker position.
