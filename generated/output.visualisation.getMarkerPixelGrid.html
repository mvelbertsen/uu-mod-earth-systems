output.visualisation.getMarkerPixelGrid¶

output.visualisation.getMarkerPixelGrid(params, markers, grid, xres)¶

maps the markers to a pixel grid, where each pixel records it’s nearest marker’s number. These marker indexes can then be used to plot a specific field from the markers.

Parameters¶

paramsParameters object

Simulation’s parameters object.

markersMarkers object

Contains all the marker values for each variable.

gridGrid object

Contains all the grid variables at the current time.

xresINT

x-direction resolution of the pixel grid, y will be set from this to match physical size aspect ratio.

Returns¶

mark_numsARRAY

Pixel grid of values of the nearest marker’s index to that pixel.
