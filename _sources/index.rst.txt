Modelling Earth Systems - Project code
======================================

This is a python reworking of the thermomechanical code developed in the book Numerical Geodynamic Modelling by Taras Gerya [1]_ . 
It uses finite differences on a staggered grid to solve the Stokes, continuity and temperature equations in 2D, with marker-in-cell adevection.




.. [1] Gerya T. MATLAB program examples. In: Introduction to Numerical Geodynamic Modelling. Cambridge University Press; 2019:425-437. `DOI <https://doi.org/10.1017/9781316534243.024>`_
